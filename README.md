# MLproject - ASL Translator

Using https://github.com/dxli94/WLASL as its data base and only using classes with more than 7 entries. 

With the fact that each class is smaller than we assumed or thought, since videos that were referenced in the dataset we found were old and some don't exist anymore. With using mediapipe, we are going to standardize the number of frames taken from each video, so the speed of someone's sign won't negatively affect our model.
